# mini-splunk-server
Concurrent Syslog Analytics Server ("Mini-Splunk") for NSAPDEV (Server Application Development) 

**To run as SERVER:**

1. run 'run_server.py'

**To run as CLIENT**

1. run 'run_client.py'

**CLIENT Command List:**

1. INGEST <FILE_PATH> <SERVER_IP>:<SERVER_PORT>
    - *Reads a local file and streams it to the server for parsing.*
        - Enter command: INGEST /var/log/syslog 192.168.1.100:65432
2. PURGE <SERVER_IP>:<SERVER_PORT>
    - *Instructs the server to safely delete all indexed log entries from its memory or disk. The server must acquire an exclusive write lock to ensure no other threads are reading or writing data while the clear operation is in progress.*
        - Enter command: PURGE 192.168.1.100:65432
3. "QUERY <SERVER_IP>:<SERVER_PORT> <SEARCH_DATE|SEARCH_HOST|SEARCH_DAEMON|SEARCH_SEVERITY|SEARCH_KEYWORD|COUNT_KEYWORD> <value>
    - <SEARCH_DATE> *Asks the server to return all parsed log lines that match a specific date or timestamp prefix.*
        - Enter command: QUERY 192.168.1.100:65432 SEARCH_DATE Feb 22
    - <SEARCH_HOST> *Asks the server to return all parsed log lines originating from a specific machine.*
        - Enter command: QUERY 192.168.1.100:65432 SEARCH_HOST SYSSVR1
    - <SEARCH_DAEMON> *Filters the server's indexed logs to show only those generated by a specific service.*
        - Enter command: QUERY logserver.local:8080 SEARCH_DAEMON sshd
    - <SEARCH_SEVERITY> *Filters the server's indexed logs to show only those matching a specific severity level (e.g., ERROR, WARN, INFO).*
        - Enter command: QUERY 192.168.1.100:65432 SEARCH_SEVERITY ERROR
    - <SEARCH_KEYWORD> *Scans the logs (specifically the "Message" portion) and returns the full log lines containing the exact keyword or phrase.*
        - Enter command: QUERY 192.168.1.100:65432 SEARCH_KEYWORD Failed password
    - <COUNT_KEYWORD> *Searches the logs and returns an aggregate count of how many times the keyword appears, rather than the log lines themselves.*
        - Enter command: QUERY 192.168.1.100:65432 COUNT_KEYWORD Deactivated
4. HELP
    - *Prints out the command list on the CLI client application.* 
        - Enter command: HELP
5. EXIT
    - *Exits the CLI client application.*
        - Enter command: EXIT
